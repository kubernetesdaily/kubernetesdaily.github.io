#### write simple c program 

```c
#include<stdio.h>

int main()
{
printf("dockerworkshop");
}
```

#### Compile C program 

```sh
gcc -o hello hello.c 

✗ ./hello 
dockerworkshop%     

```

#### create dockerfile with following content : 


```dockerfile
FROM scratch
ADD hello /
CMD ["/hello"]
```
