cat Dockerfile 

```dockerfile
FROM php:7.2-apache
COPY index.php /var/www/html

```

add index.php file  

```php
<?php

echo "hello world \n\n";
```

creat docker-compose.yaml 

```yml
version: '3'
services:
  phpapp:
    ports:
      - "8080:80"
    build:
      context: ./
      dockerfile: Dockerfile

```

to run docker compose 

```sh

docker compose up --build
```
