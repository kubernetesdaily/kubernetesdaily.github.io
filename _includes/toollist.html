Datatables Markdown Demo

Based on tutorial for Jekyll.

Steps:

  1. Add datatables CSS
    • https://cdn.datatables.net/1.10.21/css/jquery.dataTables.min.css
  2. Add any custom CSS, like font-family
  3. Add jQuery JS
    • https://code.jquery.com/jquery-3.5.1.js
  4. Add jQuery datatable JS
    • https://cdn.datatables.net/1.10.21/js/jquery.dataTables.min.js
  5. Add table (HTML, or Markdown with pre-processor). No special class is needed, you just need to reference your table next.
  6. Add JS snippet to run jQuery DataTable setup your table.

You datatable will be generated. It will include an entries limit (default 10), a search bar, sorting and pagination.
